Learning C# with Xamarin: Collections

Duration

20 minutes

Lab goals

In this exercise, you will modify an existing console application by adding an employee array, populating the array with employees, and then displaying the employee names. If you use the lab instructions, your display will look something like:

To complete this exercise, you will perform the following steps:

  1. Create a new class to model an employee, this will be a custom class named Employee that should have first and last name properties.
  2. Modify Program.cs code to create an array of Employee objects named employeeArray.
  3. Populate the employeeArray with new employee objects.
  4. Finally, display the employeeArray in the console using a for or foreach loop.
  5. There are TODO comments in the code to direct your work, except for step 1 which requires you create a new class.

Required Assets

The provided Resources folder for this part contains a subfolder named Start with a solution you will use as starter code. The Completed subfolder contains a completed project you can use to check your work.

Steps

Try to use the picture and above steps to complete the lab exercise, referring to the slides if you need any help. You can also use the following step-by-step instructions to fill in any knowledge gaps. Make sure to use the starter project located in the Resources/Start folder.

Create an Employee class

First, you need to create a new employee class.

  1. Add a new class to the EmployeeArray project named Employee.cs. Remember you do this using File > New File.. or by right-clicking on the project and selecting Add > New File.
  2. Select Empty Class from the templates (or just Class) and name it Employee.cs.
  3. Add two public properties of type string to hold the FirstName and LastName. You can delete the default constructor if you like - remember this is the constructor that takes no parameters, it was added automatically in Visual Studio, but we don't need it here.

    4. Show Code 

    namespace EmployeeArray
{
    public class Employee
    {
        public string FirstName { get; set; }
        public string LastName { get; set; }
    }
}

Create the Employee Array

Now, you will add an array of Employee objects into the Program.cs file.

  1. Open the Program.cs file in the EmployeeArray project. Notice that it has several TODO comments in the Main method, this is where you will place your work.
  2. Locate the TODO related to creating the array. Use the same syntax you used in the first exercise, except this time you want an array of Employee objects. Allocate an array of at least three (3) employees (feel free make the array larger if you would like to).

    3. Show Code 

    // TODO: Create employeeArray
Employee[] employeeArray = new Employee[3];

Populate the employee array

Next, you will create a series of employee objects to fill in the array.

  1. Populate the array with three employees, you will need to assign each index (remember it starts with zero), and provide unique first and last names for each created employee object. For example: 
    2. // TODO: Populate employeeArray with some data
employeeArray[ 0 ] = new Employee { FirstName = "Jesse", LastName = "Liberty" };
 ...

Display the employee array

Next, you will display the employees using the console

  1. Iterate over the employee array using a for or foreach loop and print each element to the console.

    2. Show Code 

    // TODO: Display employeeArray
for (int i = 0; i < employeeArray.Length; i++)
{
     Console.WriteLine(employeeArray[i]);
}

  2. Run the application to see the results.
  3. Did it do what you expected? If you've followed along with the instructions here, it should look something like:
  4. This isn't displaying the values like it did before, and that's because C# doesn't know how to display an Employee object - you have to tell it by adding another method named ToString().
  5. Open the Employee.cs file you created earlier. At the bottom of the class, add the following code to tell C# how to display your employee: 
    6. public class Employee
{
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public override string ToString ( )
    {
        return string.Format( "{0} {1}", FirstName, LastName );
    }
}
  6. Run the application again to see the difference - it should now be displaying your employee names correctly!

Summary

In this lab exercise, you modified the EmployeeArray application by adding a new employee array, populating the array with new employee data and then displaying that data. Next you will define a List of employee objects and populate that list.
