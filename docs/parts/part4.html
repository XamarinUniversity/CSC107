Learning C# with Xamarin: Collections

Duration

15 minutes

Lab goals

In this final exercise, you will create a dictionary that associates Employees with their integer IDs. To complete the exercise, you will perform the following steps:

  1. Create a Dictionary to map from int to Employee.
  2. Add three employees using the IDs 101, 102 and 103 to the dictionary.
  3. Pull each employee out of the dictionary by the ID.
  4. Display each employee in the console.

Define a Dictionary to hold employees

You need to hold a group of employees and be able to look them up quickly using their employee ID. While you could use a List or array, neither provides the "lookup by ID" feature so you would have to search to find each employee which is inefficient. Instead, you will use a Dictionary which provides faster lookup without the need to search.

  1. Open the EmployeeDictionary.sln solution from the Resources/Start folder. This project has the Employee definition.
  2. Open the Program.cs file in the EmployeeDictionary project. It has some existing code and a set of TODO comments where you will add some code.
  3. After the first TODO comment, create a Dictionary that maps from int to Employee. Remember this is defined as Dictionary<int,Employee>. Name it employeeDictionary.

    4. Show Code 

    // TODO: create a Dictionary from int to Employee
Dictionary<int,Employee> employeeDictionary = new Dictionary<int,Employee>();

  4. Add three employees using the integer IDs 101, 102 and 103 to the dictionary, For example, here is one created and using the ID 101: 
    5. // TODO: Add an employee with an id of 101 to the dictionary
employeeDictionary.Add( 101,
  new Employee {
      Id = 101,
      FirstName = "Jesse",
      LastName = "Liberty",
  } );
  5. Next, index into the dictionary using the Id to get each employee and then print them into the console.
  6. Run the application to see the results!

Summary

In this lab exercise, you created a dictionary mapping an integer ID to an employee object, added three employee, and then printed them into the console.
