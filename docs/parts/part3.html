Learning C# with Xamarin: Collections

Duration

10 minutes

Lab goals

In this exercise, you will use a List to hold a group of employee objects and then display them in the console. If you use the lab instructions, your display will look something like:

To complete the exercise, you will perform the following steps:

  1. Create a new List to hold Employee objects.
  2. Add some employees, populating them with a first and last name.
  3. Display the list of employees inside the console

Required Assets

The provided Resources folder for this part contains a subfolder named Start with a solution you will use as starter code. The Completed subfolder contains a completed project you can use to check your work.

Steps

Try to use the picture and above steps to complete the lab exercise, referring to the slides if you need any help in creating the List. You can also use the following step-by-step instructions to fill in any knowledge gaps.

Define a list of employees

This will be very similar to what you did in the last exercise, except you will use a List instead of the array. The Employee class is already provided and it already has the ToString method you used last time.

  1. Open the Program.cs file in the ListOfEmployees project. As before, it has several TODO comments where you will place your code.
  2. As a first step, add a List to hold Employee objects. Remember this is defined as a List<Employee>.
  3. Add a few employees to the list using the list's Add method. Provide some unique first and last names as you add them.
  4. Finally, using a for or foreach loop, display the elements of the list into the console.

    5. Show Code 

    // TODO: define a List of employee objects
List<Employee> employeeList = new List<Employee>();
// TODO: populate some employees into your list
employeeList.Add(new Employee { FirstName = "Jesse", LastName = "Liberty" });
employeeList.Add(new Employee { FirstName = "George", LastName = "Washington" });
employeeList.Add(new Employee { FirstName = "Mark", LastName = "Smith" });
// TODO: display the list
for (int i = 0; i < employeeList.Count; i++)
{
     Console.WriteLine(employeeList[i]);
}

  5. Run the application to see the results - it should look exactly like the array version did, but now you can add as many employees without redefining the size of the list.

Summary

Congratulations! In this lab exercise, you modified the ListOfEmployees application by adding a list of employees. Next you will create a Dictionary from int to Employee, add employee IDs to the Dictionary and add text for the labels.
