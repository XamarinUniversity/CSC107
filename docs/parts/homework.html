30 minutes

Homework goals

Here you will write a console version of the classic game of hangman. Sample output from the completed application looks like this, notice the hidden word being revealed as you guess correct letters.

You will need to perform the following steps:

  1. Build a HangmanManager class to manage the entire game. This includes:
    • A private field to store the hidden word.
    • A public read only auto property to store the string of correctly guessed characters and asterisks.
    • A public read only property to indicate if the game is over.
    • A constructor that accepts the hidden word to display.
    • A public method to make a guess.

    Hint 1: The constructor is responsible for creating the string that contains the asterisks. Remember that all strings have a Length property that returns the number of characters in the string. Consider using this with a loop.

    Hint 2: All strings have a Remove and Insert method. Consider using them in your Guess method to update the string that contains the asterisks.

    Hint 3: A string is just an array of characters.

  2. Create a HangmanManager variable in the Main method.
  3. Instantiate your HangmanManager variable.
  4. Create a game loop that continues to execute as long as the game is not over.

    5. Hint: There's a property on your HangmanManager that will tell you when the game is over.

  5. Each time the loop executes, display the string that has the correctly guessed characters and asterisks.
  6. Prompt the user to enter a single character.
  7. Call the Guess method on your HangmanManager passing in the guessed character.
  8. If the game is over, display a winner message.

Solution
