Before you begin

This is a Xamarin cross-platform class about C#. You can use either Visual Studio for Mac or Visual Studio for Windows. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this course, you will store groups of things (i.e. Collections) in memory using three techniques: array, List, and Dictionary. You will create each type of collection and load it with data. In each case, you will use a console app to display the output.

Objectives

  1. Illustrate and explain how to use arrays
  2. Utilize Lists and Dictionaries

Lecture

Learning C# with Xamarin: Collections

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Creating and Displaying arrays
    Exercise
    In this exercise, you will create an array of integers, fill it with data, and display it to the user.
  2. Create array of Employees
    Exercise
    Here you will create an array of employees, populate the array with data, and display it to the user.
  3. Create a List of Employees
    Exercise
    In this exercise, you will create a List of Employees, populate it with employee objects, and display it to the user.
  4. Create a Dictionary keyed by EmployeeID
    Exercise
    In this final exercise, you will create a Dictionary to associate each Employee with an integer Id, load employees into the dictionary, and display it to the user.
